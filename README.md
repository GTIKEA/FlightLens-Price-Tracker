# ✈ FlightLens — Flight Price Intelligence Dashboard

A modern real-time flight price analytics dashboard built using Python, SQL, and interactive frontend visualizations.

## 🚀 Features

- Real-time flight price tracking
- Interactive dashboards
- Airline comparison analytics
- Booking heatmaps
- SQL-powered insights
- Responsive enterprise-grade UI
- Dark/Light theme support
- Export functionality

## 🛠 Tech Stack

- Python
- MySQL
- HTML/CSS/JavaScript
- Chart.js
- BeautifulSoup
- SerpAPI
- Tableau

## 📊 Dashboard Modules

- Price Trends
- Booking Advance Analysis
- Airline Market Share
- Day-wise Pricing
- Heatmaps
- Data Tables
- Price Alerts

## 📁 Project Structure

```bash
frontend/   → Main dashboard
docs/       → Full implementation guide
sql/        → Database schema
python/     → Data collection scripts
screenshots/→ Dashboard previews
```

## ⚡ Setup

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/FlightLens-Price-Tracker.git
```

### Open Dashboard

```bash
cd frontend
```

Open:

```bash
flight_price_tracker.html
```

in browser.
